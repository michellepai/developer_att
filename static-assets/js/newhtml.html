Send SMS

Select provider:

  • AT&T
  • GSMA
Overview

The Send SMS method sends an SMS message to one or more AT&T Wireless network mobile devices. The SMS messages are processed synchronously and sent asynchronously to a destination on the AT&T Wireless network. A unique identifier (ID) is returned in the response that may be used to query the status of the message that has been sent.

								OutboundSMSResponse Object

                                messageId
                                
								String
								Body
                              

                            

                            

                            Specifies the unique identifier for the message. This parameter value has two different purpose based on the scenario. If the notifyDeliveryStatus parameter is either set to false or not present in the request, then this parameter value is used as a parameter in the URI on the Get MMS Delivery Status method to obtain the delivery status of the message. If the notifyDeliveryStatus parameter is set to true in the request, then this parameter value is be returned in the delivery status callback notification sent to the registered listener for the app. This parameter is a child of the outboundSMSResponse parameter.
                            

                        

                        
						

                            

                              

                                resourceReference
                                
								ResourceReference Object
								Body
                              

                            

                            

                            Specifies the location of the created resource. This parameter is a child of the outboundSMSResponse parameter.
                            

                        

                        
						

                            

                              

                                resourceUrl
                                
								String
								Body
                              

                            

                            

                            Specifies the URI that provides network delivery status of the sent message. This parameter is the only child of the ResourceReference parameter.
